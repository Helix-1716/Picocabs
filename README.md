# Cab Page | Admin Operations Dashboard

A premium, European-style SaaS dashboard for real-time ride-hailing operations.

## 🚀 Technical Stack
- **Framework**: React 18 + Vite + TypeScript
- **Styling**: Tailwind CSS (European Minimalist Theme)
- **Animations**: GSAP (Technical Precision feel)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Navigation**: React Router 6

## 🛠️ Key Features
1. **Fleet Overview**: KPI grid with real-time revenue and ride tickers.
2. **SOS Command Center**: Mission-critical "War Room" for emergency response with tactical map mocks.
3. **Driver Management**: High-density data tables for fleet compliance and monitoring.
4. **European Aesthetic**: Deep navy/black base, minimal borders, and professional typography.

## 🏁 Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
